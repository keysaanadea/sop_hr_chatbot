Overview

Apa itu DENAI?

DENAI adalah Enterprise Knowledge Assistant yang memungkinkan karyawan mendapatkan informasi perusahaan secara instan — baik lewat teks maupun suara.


Sistem ini memiliki dua jalur utama: menjawab pertanyaan tentang SOP & kebijakan perusahaan dari dokumen PDF, dan menganalisis data HR langsung dari database PostgreSQL.

Core Features

Dua Mode Interaksi

💬

Chat Mode

Antarmuka chat teks real-time dengan streaming SSE

  • Streaming jawaban token per token
  • Tabel data HR interaktif + Chart.js
  • Riwayat sesi multi-percakapan
  • Feedback thumbs up/down per jawaban
  • Role-based access (Employee vs HR)
  • Markdown rendering otomatis
📞

Call Mode

Overlay telepon full-screen — seperti menelepon AI

  • Rekam suara via MediaRecorder API
  • Transkripsi real-time (OpenAI Whisper)
  • AI bicara balik via ElevenLabs TTS
  • Loop otomatis: dengar → proses → jawab
  • Transcript muncul di layar
  • Jawaban dikompres max 150 token
